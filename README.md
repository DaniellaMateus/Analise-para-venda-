# Analise-para-venda-
Este projeto tem como objetivo realizar uma análise detalhada de vendas e desempenho de lojas para apoiar a tomada de decisão estratégica do Sr. João, proprietário de uma rede varejista. A análise busca identificar qual loja apresenta o menor desempenho, de modo que ela possa ser vendida e o capital reinvestido em um novo negócio.
